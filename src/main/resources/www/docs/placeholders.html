Placeholders

You can use placeholders in the HTML file to display dynamic information. Placeholders are enclosed in double curly braces and are replaced with the actual value when the HTML file is parsed.

Placeholders List

  • renderPartial("partial/header.html"): render a partial HTML file.
  • getPathFor("file.txt"): display the path for a file.
  • {{variable}}: display the variable set in the context at the controller level.

Context

Context can be used at the controller level to pass data to the HTML when rendered. For example, to pass a variable to the HTML, you can do the following: 
context.put("variable", "value");
Then, in the HTML file, you can use the variable like this: 
{{variable}}
