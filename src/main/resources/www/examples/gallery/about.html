In this example, JAWS is using API calls to get the images from the backend. This API is also managed by JAWS, and it is a simple GET request that returns a JSON response.

The API is defined in the GalleryAPIController class, and it is used in the GalleryController class to get the images from the database.

The API is organized with the classic folder structure, with a controller, service, repository

A single DTO class is used to represent the image data, and it is defined in the dto package.

The API has 2 endpoints:

  • /api/v1/gallery to get all the images.
  • /api/v1/gallery/:id to get an image by its ID.

Its possible to use authorization to restrict access to the API endpoints, using the @AccessControl annotation.
