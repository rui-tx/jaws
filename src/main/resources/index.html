JAWS - Just Another Web Server

This is the default page for JAWS.

You can change this page by modifying the index.html file in the resources folder.

If you want to select a directory to serve, use the Environment Variable WWWPATH with the path to the directory.

Placeholders

You can use placeholders in the HTML file to display dynamic information.

Placeholders are enclosed in double curly braces, and they are replaced with the actual value when the HTML file is parsed.

For example, you can use the placeholder { { getServerPort } } (without the spaces) to display the server port.

Placeholders List

  • getPathFor("file.txt"): display the path for a file.
  • getServerPort: display the server port.
  • getCurrentConnections: display the current number of connections.
